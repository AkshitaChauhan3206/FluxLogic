# 🚀 FluxLogic AI: Your Intelligent Business Dashboard

FluxLogic is an advanced, AI-powered intelligence platform designed to turn your raw data into actionable business strategies. Whether you are tracking sales, customer behavior, or operational trends, FluxLogic provides a "360-degree view" of your business health.

---

## ✨ Key Features

*   **📊 Dynamic Intelligence Dashboard**: Automatically generates professional charts and AI insights from your uploaded CSV data.
*   **🤖 AI Business Assistant**: A built-in chatbot that understands your specific data. Ask it about profits, trends, or for strategic advice!
*   **🛡️ Enterprise-Grade Security**: Protect your data with Two-Factor Authentication (2FA) via Authenticator apps or Email.
*   **🔒 Local & Private**: Your data is encrypted and stays on your machine—never shared with external clouds.
*   **🎨 Personalized Experience**: Choose your theme (Dark/Light) and adjust font sizes for comfortable viewing.

## 🛠️ How to Get Started (Simplified)

### 1. Prerequisites
You need **Python** installed on your computer. If you don't have it, download it from [python.org](https://www.python.org/).

### 2. Set Up the Project
Open your terminal (Command Prompt on Windows, Terminal on Mac) and follow these simple steps:

```bash
# Install all required components
pip install -r requirements.txt
```

### 3. Run the Application
Start the engine with this command:
```bash
python app.py
```

### 4. Open in Your Browser
Once the command is running, open your web browser and go to:
**[http://127.0.0.1:5000](http://127.0.0.1:5000)**

## 📖 Quick Start Guide

1.  **Register/Login**: Create a secure account to keep your data private.
2.  **Upload Data**: Go to the "Datasets" section and upload any **CSV file** (like a sales report or inventory list).
3.  **Explore Analytics**: Click "Options" on your dataset to view the **Analytics Dashboard**.
4.  **Chat with AI**: Need specific answers? Click "AI Assistant" and ask things like:
    *   *"What was my total revenue last month?"*
    *   *"Which category is growing the fastest?"*
    *   *"What are the top 3 items I should focus on?"*

## ❓ Frequently Asked Questions

**Q: Is my data safe?**
**A:** Yes! FluxLogic uses AES-256 encryption. Your datasets are encrypted the moment they touch the server.

**Q: What file types are supported?**
**A:** Currently, FluxLogic is optimized for **.csv** files. Make sure your file has clear column headers (like "Date", "Sales", "Category").

**Q: I see a 2FA setup screen, what is that?**
**A:** This is an extra layer of security. You can use apps like Google Authenticator or Microsoft Authenticator to link your account.

## 🔧 Technical Details (For Developers)
*   **Backend**: Flask (Python)
*   **Database**: SQLite with SQLAlchemy
*   **Frontend**: Vanilla HTML5, CSS3 (Modern Glassmorphism Design)
*   **AI/ML**: Scikit-learn for intent detection, Pandas for data processing
*   **Visualization**: Chart.js
